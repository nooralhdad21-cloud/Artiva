## 🌟 المميزات الرئيسية

* **حفظ الصور:** إمكانية تحميل البطاقة كصورة عالية الجودة (PNG) بضغطة زر.
* **جاهز للطباعة:** تنسيق مخصص للطباعة الورقية المباشرة.
* **متجاوب بالكامل:** يعمل بسلاسة على كافة أنواع الهواتف الذكية والمتصفحات.

## 🛠️ التقنيات المستخدمة
تم بناء هذا المشروع باستخدام أحدث تقنيات الويب الأساسية لضمان السرعة والخفة:
* **HTML5:** لبناء الهيكل البرمجي.
* **CSS3:** للتصاميم المتجاوبة والأنماط البصرية الحديثة.
* **JavaScript (Vanilla):** لإدارة المنطق البرمجي والمعاينة اللحظية.
